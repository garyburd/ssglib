ssglib is a Lua library for building static websites from [Obsidian](https://obsidian.md) vaults using [Pandoc](https://pandoc.org).

Features:

- Read and cache Obsidian vault metadata
- Resolve wikilinks and process callout blocks
- Generate HTML with a programmatic element builder
- Manage output directories with incremental updates
